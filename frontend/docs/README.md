## VS Code 插件

在 day-primer-web 的开发过程中使用了以下 VS Code 插件：

| 插件 ID | 功能 | 配置文件 |
| --- | --- | --- |
| streetSideSoftware.code-spell-checker | 拼写检查 | cspell.json |

推荐的 VSCode 配置如下：

``` json
{
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  }
}
```
